# MyDashboard
Personal university dashboard with everything that could be important.

- website shortcuts
- exam schedule
- TUHH website search
- ... more to come

[Demo]

### Tech
* [Twitter Bootstrap 4] - wanted to try out Bootstrap 4 ...
* [vue.js] - .. and also this
* [Google Charts] - used for the timeline (exam schedule)
